# 프로젝트 로드맵

## Phase 0: 환경 세팅
- **목표**: 프로젝트 초기 환경 구성, 인증 시스템 구현, DB 스키마 생성
- **기간**: 1-2일
- **포함 기능**: F-01 (회원가입/로그인), F-03 (PIN 잠금)
- **병렬 그룹**: PG-0A (F-01, F-03)
- **주요 작업**:
  - Next.js 14 + TypeScript + Tailwind CSS 프로젝트 초기화
  - Supabase 프로젝트 생성 및 연동
  - 전체 DB 스키마 생성 (users, todos, routines, transactions 등 전체 테이블)
  - Row Level Security (RLS) 정책 적용
  - Supabase Auth 기반 이메일/Google OAuth 인증 구현
  - PIN 잠금 시스템 구현
  - shadcn/ui 설치 및 기본 컴포넌트 설정
  - Zustand + React Query 설정
- **완료 기준**:
  - 이메일 회원가입/로그인 정상 동작
  - Google OAuth 로그인 정상 동작
  - RLS 적용된 DB에서 본인 데이터만 조회 확인
  - PIN 설정/인증/잠금 동작 확인

## Phase 1: 대시보드 및 기반 UI
- **목표**: 앱 레이아웃, 네비게이션, 메인 대시보드 구축
- **기간**: 2-3일
- **포함 기능**: F-02 (메인 대시보드), F-04 (PWA 지원)
- **병렬 그룹**: PG-1A (F-04) -- F-02는 의존성으로 단독 진행
- **주요 작업**:
  - 앱 공통 레이아웃 (헤더, 하단 네비게이션 바, 사이드바)
  - Mobile-First 반응형 레이아웃 구현
  - 메인 대시보드 페이지 - 4개 모듈 요약 카드
  - 빠른 입력 FAB(Floating Action Button) 구현
  - PWA manifest.json 및 Apple 메타태그 설정
  - 앱 아이콘 생성 및 적용
- **완료 기준**:
  - 대시보드에서 4개 모듈 요약 카드 표시 (데이터는 더미/빈 상태)
  - 하단 네비게이션으로 모듈 간 이동 가능
  - iOS Safari 홈 화면 추가 시 standalone 모드 실행
  - 모바일(375px) 레이아웃 깨짐 없음

## Phase 2: 시간 관리 모듈
- **목표**: 할일 관리, 캘린더 뷰, 루틴 시스템 완성
- **기간**: 3-4일
- **포함 기능**: F-05 (할일 관리), F-06 (캘린더 뷰), F-07 (루틴 설정/체크인), F-20 (완료율 통계)
- **병렬 그룹**:
  - PG-2A: F-05, F-07 (독립 CRUD, 동시 개발 가능)
  - PG-2B: F-20 (F-05, F-07 완료 후)
- **주요 작업**:
  - Todo CRUD (생성/조회/수정/삭제/완료 토글)
  - 할일 드래그 정렬
  - 일간/주간/월간 캘린더 뷰 구현
  - 루틴 생성 (반복 주기, 시간 설정)
  - 루틴 체크인 UI 및 streak 카운터
  - 주간/월간 완료율 차트 (Recharts)
- **완료 기준**:
  - 할일 CRUD 전체 동작
  - 캘린더에서 할일 확인 및 날짜 클릭 상세 조회
  - 루틴 생성/체크인/streak 정상 동작
  - 완료율 차트 렌더링 확인

## Phase 3: 금전 관리 모듈
- **목표**: 수입/지출 관리, 예산 시스템, 지출 분석 차트 완성
- **기간**: 3-4일
- **포함 기능**: F-08 (수입/지출), F-09 (카테고리 관리), F-10 (예산 설정), F-11 (지출 대시보드), F-21 (자산 현황), F-22 (월별 지출 추이)
- **병렬 그룹**:
  - PG-3A: F-09, F-10 (독립 설정 기능, 동시 개발 가능)
  - PG-3B: F-21, F-22 (독립 조회 기능, 동시 개발 가능)
- **주요 작업**:
  - 카테고리 관리 (기본 카테고리 + 커스텀)
  - 수입/지출 CRUD + 즐겨찾기
  - 월별 예산 설정 및 초과 경고
  - 지출 현황 대시보드 (파이 차트, 바 차트)
  - 자산 현황 수동 입력 및 요약
  - 6개월 지출 추이 라인 차트
- **완료 기준**:
  - 수입/지출 입력/수정/삭제/조회 정상 동작
  - 카테고리 CRUD 및 커스텀 카테고리 추가
  - 예산 설정 후 초과 시 시각적 경고 표시
  - 파이 차트/바 차트/라인 차트 정상 렌더링

## Phase 4: 건강 트래킹 모듈
- **목표**: 식단/음주/수면/운동 기록 및 건강 대시보드 완성
- **기간**: 4-5일
- **포함 기능**: F-12 (식사 기록), F-13 (음주 기록), F-14 (수면 기록), F-15 (건강 대시보드), F-23 (체중/체성분), F-24 (운동 기록), F-25 (음주 경고)
- **병렬 그룹**:
  - PG-4A: F-12, F-13, F-14 (독립 기록 기능, 동시 개발 가능)
  - PG-4B: F-23, F-24, F-25 (독립 기록/경고 기능, 동시 개발 가능)
- **주요 작업**:
  - 식사 기록 CRUD (식사 유형, 음식명, 칼로리)
  - 음주 기록 CRUD (주종, 도수, 양, 잔 수)
  - 수면 기록 (취침/기상 시각, 수면 질)
  - 건강 대시보드 요약 카드
  - 체중/체성분 기록 및 추이 차트
  - 운동 기록 CRUD
  - WHO 기준 음주 경고 배너
- **완료 기준**:
  - 식사/음주/수면 기록 CRUD 전체 동작
  - 건강 대시보드에 오늘/주간 요약 정상 표시
  - 체중 추이 차트 렌더링
  - 음주 기준 초과 시 경고 배너 표시

## Phase 5: 사적 기록 모듈
- **목표**: 일기 시스템, 감정 추적, 관계 메모 및 암호화 완성
- **기간**: 4-5일
- **포함 기능**: F-16 (일기 작성), F-17 (감정 캘린더), F-18 (사적 기록 PIN 잠금), F-26 (일기 검색), F-27 (인간관계 메모)
- **병렬 그룹**:
  - PG-5A: F-16, F-18 (일기 작성과 PIN 잠금, 동시 개발 후 통합)
  - PG-5B: F-26, F-27 (독립 조회/기록 기능, 동시 개발 가능)
- **주요 작업**:
  - 사적 기록 라우트 PIN 미들웨어 적용
  - 리치 텍스트 일기 에디터 구현
  - 감정 태그 선택 UI (10개 감정 아이콘)
  - 클라이언트 사이드 AES-256 암호화/복호화 구현
  - 감정 캘린더 (월간 달력 + 감정 아이콘)
  - 일기 검색 (복호화 후 클라이언트 검색)
  - 인간관계 메모 CRUD + 암호화
- **완료 기준**:
  - PIN 인증 없이 /private 접근 불가
  - 일기 작성/조회/수정 정상 동작 (암호화 확인)
  - 감정 캘린더에 감정 아이콘 정상 표시
  - 일기 키워드 검색 및 태그 필터 동작
  - 인간관계 메모 CRUD + 암호화 저장 확인

## Phase 6: 통합 리포트 및 마무리

- **기간**: 3-4일
- **포함 기능**: F-19 (주간/월간 리포트), F-28 (다크 모드), F-29 (시간 블록), F-30 (정기 지출), F-31 (식단 목표), F-32 (감정 통계), F-33 (데이터 내보내기)
- **병렬 그룹**:
  - PG-6A: F-19 (통합 리포트, 단독)
  - PG-6B: F-28, F-29, F-30 (독립 기능, 동시 개발 가능)
  - PG-6C: F-31, F-32, F-33 (독립 기능, 동시 개발 가능)
- **주요 작업**:
  - 4개 모듈 통합 주간/월간 리포트 자동 생성

  - CSV/JSON 데이터 내보내기
  - Lighthouse 성능 최적화
  - Vercel 배포 및 커스텀 도메인 설정
- **완료 기준**:
  - 주간/월간 리포트 자동 생성 및 조회 가능
  - 다크 모드 전체 페이지 적용
  - P2 기능 정상 동작
  - Lighthouse Performance Score 80점 이상
