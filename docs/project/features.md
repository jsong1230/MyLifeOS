## 병렬 그룹 규칙

- 같은 마일스톤 내에서만 그룹 구성
- 그룹 내 기능 간 상호 의존성 없음
- 그룹 내 기능 간 충돌 영역 미겹침
- 그룹당 최대 3개 기능

## 상태 범례

## 인수조건 (Acceptance Criteria)

### F-01: 회원가입/로그인
**모듈**: 공통
**우선순위**: P0
**설명**: 이메일 또는 Google OAuth 소셜 로그인 지원. Supabase Auth 기반.

- [ ] AC-01: 이메일과 비밀번호로 회원가입 시 인증 이메일 발송 후 계정 활성화
- [ ] AC-02: Google OAuth를 통한 소셜 로그인으로 계정 생성 및 즉시 로그인
- [ ] AC-03: 로그인 성공 시 JWT 세션 생성 및 메인 대시보드로 리다이렉트
- [ ] AC-04: 비밀번호 분실 시 이메일 기반 비밀번호 재설정 가능
- [ ] AC-05: 30분 비활동 시 자동 로그아웃 처리

### F-02: 메인 대시보드
**모듈**: 공통
**우선순위**: P0
**설명**: 4개 모듈의 오늘/이번주 요약 카드 표시, 빠른 입력 버튼 제공.

- [ ] AC-01: 시간 모듈 요약 카드 - 오늘 할일 완료/전체 수, 오늘 루틴 달성 현황 표시
- [ ] AC-02: 금전 모듈 요약 카드 - 이번달 총 지출, 예산 대비 소진률 표시
- [ ] AC-03: 건강 모듈 요약 카드 - 오늘 칼로리 섭취, 이번주 음주 횟수, 어젯밤 수면 시간 표시
- [ ] AC-04: 사적 기록 요약 카드 - 오늘 감정 상태, 최근 일기 작성일 표시
- [ ] AC-05: 빠른 입력 버튼(FAB)으로 할일/지출/식사/일기 입력 화면에 1탭으로 진입

### F-03: PIN 잠금
**모듈**: 공통
**우선순위**: P0
**설명**: 앱 재진입 시 또는 사적 기록 접근 시 4-6자리 PIN 인증.

- [ ] AC-01: 최초 설정 시 4-6자리 PIN 등록 (확인 입력 필수)
- [ ] AC-02: 앱 재진입(백그라운드 복귀 포함) 시 PIN 입력 화면 표시
- [ ] AC-03: PIN 5회 연속 오입력 시 10분간 앱 잠금 및 경고 메시지 표시
- [ ] AC-04: PIN 변경 기능 제공 (기존 PIN 확인 후 새 PIN 설정)

### F-04: PWA 지원
**모듈**: 공통
**우선순위**: P0
**설명**: manifest.json + Apple 메타태그 설정으로 iOS Safari 홈 화면 추가 시 네이티브 앱처럼 실행.

- [ ] AC-01: manifest.json에 앱 이름, 아이콘(192x192, 512x512), 테마 색상, display: standalone 설정
- [ ] AC-02: Apple 전용 메타태그(apple-mobile-web-app-capable, status-bar-style) 적용
- [ ] AC-03: iOS Safari에서 '홈 화면에 추가' 시 주소창 없이 전체 화면으로 실행
- [ ] AC-04: 앱 아이콘이 홈 화면에 정상 표시

### F-05: 할일(Todo) 관리
**모듈**: 시간
**우선순위**: P0
**설명**: 제목, 마감일, 우선순위, 카테고리 입력. 완료 체크. 드래그 정렬.

- [ ] AC-01: 할일 생성 시 제목(필수), 마감일, 우선순위(높음/중간/낮음), 카테고리 입력 가능
- [ ] AC-02: 할일 목록에서 완료 체크박스 토글로 상태 변경 (완료/미완료)
- [ ] AC-03: 드래그 앤 드롭으로 할일 순서 변경 가능
- [ ] AC-04: 할일 수정 및 삭제 가능 (삭제 시 확인 다이얼로그)
- [ ] AC-05: 마감일 지난 할일에 시각적 경고(빨간색 표시) 적용

### F-06: 캘린더 뷰
**모듈**: 시간
**우선순위**: P0
**설명**: 일간/주간/월간 캘린더에서 할일 및 일정 확인. 날짜 클릭으로 상세 조회.

- [ ] AC-01: 월간/주간/일간 뷰 전환 가능
- [ ] AC-02: 캘린더 날짜에 해당일 할일 개수 또는 제목 미리보기 표시
- [ ] AC-03: 날짜 클릭 시 해당일 할일/루틴 상세 목록 표시
- [ ] AC-04: 캘린더에서 직접 새 할일 추가 가능 (날짜 선택 상태에서)

### F-07: 루틴 설정 및 체크인
**모듈**: 시간
**우선순위**: P0
**설명**: 반복 루틴 생성(매일/특정 요일/주기), 완료 체크, 연속 달성일(streak) 표시.

- [ ] AC-01: 루틴 생성 시 제목, 반복 주기(매일/특정 요일/N일 간격), 실행 시간 설정
- [ ] AC-02: 오늘의 루틴 목록에서 완료 체크 가능
- [ ] AC-03: 연속 달성일(streak) 카운터 표시 및 중단 시 리셋
- [ ] AC-04: 루틴 수정/삭제/일시 중지 가능

### F-08: 수입/지출 입력
**모듈**: 금전
**우선순위**: P0
**설명**: 금액, 카테고리, 메모, 날짜 입력. 자주 쓰는 항목 즐겨찾기.

- [ ] AC-01: 수입/지출 구분 선택 후 금액(필수), 카테고리, 메모, 날짜 입력
- [ ] AC-02: 자주 사용하는 항목(카테고리+금액 조합)을 즐겨찾기로 등록
- [ ] AC-03: 즐겨찾기 항목 선택 시 자동 입력으로 빠른 기록
- [ ] AC-04: 거래 목록에서 수정/삭제 가능
- [ ] AC-05: 날짜별/카테고리별 필터링 기능

### F-09: 카테고리 관리

**우선순위**: P0

- [ ] AC-01: 기본 카테고리(식비, 교통, 여가, 쇼핑, 의료, 교육, 주거, 기타) 초기 제공
- [ ] AC-02: 커스텀 카테고리 추가 시 이름과 아이콘/색상 설정
- [ ] AC-03: 카테고리 수정 및 삭제 가능 (사용 중인 카테고리는 삭제 경고)

### F-10: 월별 예산 설정

**우선순위**: P0

- [ ] AC-02: 예산 대비 현재 지출률을 프로그레스 바로 시각화
- [ ] AC-03: 예산 80% 초과 시 주황색 경고, 100% 초과 시 빨간색 경고 표시

- [ ] AC-03: 카테고리별 예산 대비 지출 달성률 바 차트 표시
